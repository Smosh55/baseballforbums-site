Format

7-Inning Games

11 players per side. Everyone bats, even if rotating in and out on defense. Single-elimination playoffs cap the season.

Standings

Run Differential

Standings are determined by run differential. Tiebreakers go to head-to-head record. Both scores vs. a double-opponent are averaged.

Rainouts

Tuesday Makeups

Any rained-out game is rescheduled for the following Tuesday, or the next available field date if the field is unavailable.
